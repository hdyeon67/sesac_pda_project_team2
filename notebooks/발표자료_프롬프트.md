# 스타티(Startie) 발표자료 생성 프롬프트 (강사 피드백 반영본)

## 프롬프트 본문
```text
당신은 투자자 대상 발표 자료를 만드는 데이터 스토리텔링 전문가입니다.
아래 조건을 반드시 반영해 슬라이드 문구를 작성하세요.

[핵심 메시지]
- "가치가 검증된 유입 채널을 찾았고, 현재 예산 100%를 그 채널에 재배분해 성장을 가속하겠다."

[분석 전제]
- 분석 기준: 2025-03-01 ~ 2025-05-31 (3개월 스냅샷)
- 사업 단계: 런칭 10개월 초기 단계
- 전략 초점: AARRR 전체가 아니라 Acquisition(유입) 중심
- 이유: 초기 단계에서는 유입 모수 확대가 MAU/매출 성장의 선행조건

[반드시 포함할 분석 흐름]
1) Step1: 채널 유입 규모 + MAU/총매출 성장 추이
2) Step2~4: 전환/매출/리텐션 진단
3) Step5: 모델A v2 (ROI + CVR + 총매출 + 볼륨 + 리텐션)
4) Step8: 집행 Top3 채널 코호트 분석 (organic/content_marketing 제외)
5) Step9: 현재 예산 100% 재배분안

[핵심 설명 조건]
- organic 점수 상승 이유를 "볼륨/총매출 기여" 관점으로 설명
- 단, 예산 집행에서는 organic 0%, content_marketing 2%, referral 12%를 고정
- BAU 급락 방지 관점에서 meta_ads 10%, instagram_influencer 8%를 유지
- 남은 예산 68%는 google_ads / youtube에 배분(최종: google_ads 38.7%, youtube 29.3%)

[출력 형식]
- 슬라이드 번호
- 슬라이드 제목
- 본문(3문장 이내)
- 핵심 수치(최대 3개)
- 발표 멘트(1줄)

[금지]
- 활성화율 중심 스토리
- 과장 문구
- 데이터 근거 없는 주장
```
